date: 2017-11-09 22:35:25

今天搭建hexo博客，在最后一切准备就绪，部署到github上却找不到页面，最终搜索处理方法，得到了解决方法。主要注意以下几点：首先注意仓库名和github的用户名一致；其次注意配置vim _config.yml文件时，

deploy:
type: git
repo: https://github.com/"yourgithubname"/"yourgithubname".github.io.git
branch: master

type后面跟git而不是github，每个冒号后面有半角的空格。
